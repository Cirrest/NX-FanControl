# NX-FanControl 21.0+

此版本采用最新的 libnx 4.10.0 编译，以符合 21.0.0 引入的 ABI/TLS 变更。

This release was compiled with the latest libnx 4.10.0, to comply with the ABI/TLS changes introduced in 21.0.0.

### Requirements

[devkitPro](https://devkitpro.org/wiki/Getting_Started)

### Building from source

```
git clone --recurse-submodules https://github.com/Zathawo/NX-FanControl.git
cd NX-FanControl
make all
```
